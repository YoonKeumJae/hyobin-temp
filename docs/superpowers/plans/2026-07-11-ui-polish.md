# UI Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the app feel like a clean, warm, and cute parenting app with softer spacing, rounded cards, and a more polished top-left settings menu.

**Architecture:** Keep the current single-page flow, but introduce a consistent visual shell around each screen so the UI feels intentional. Use shared styling patterns for cards, primary actions, and menu surfaces instead of ad hoc utility combinations scattered through the page.

**Tech Stack:** React 19, TypeScript, Vite, Tailwind CSS utilities, existing frontend build (`npm run build`).

---
